    box-sizing

  1. The box-sizing property in CSS is used to control how the total width and height of an element are calculated, taking into account the padding and border of the element. It allows you to specify whether an element's dimensions should include its padding and border or exclude them from the calculation.

  2. content-box:-

    It is a default box-model where padding height and width is according to the content we have....it includes only content....means that we have content area other than padding margin and border....it does not include border and padding

  3. border-box:

    When we include border-box we have border , margin and padding inclusive in the overall dimension i.e, unlike content-box it has content=content+margin+border+padding.....

list-style

  1. list-style property is used to control the appearance of list items
  2. The list-style property can take three values:
    1. list-style-type: This value is used to set the type of marker/bullet point for the list items eg- none,disc,circle etc....
    2. list-style-position: This value is used to set the position of the marker relative to the list item's content.
    3. list-style-image: This value allows you to specify a custom image as the marker for the list items. You can use a URL to an image file to set this value.

    text-decoration

  1. the text-decoration property is used to control the decoration of text, such as underlines, overlines, line-throughs, and blinking effects. It allows you to style the appearance of text to add visual emphasis or decoration.

:root in css

  1. In CSS (Cascading Style Sheets), the :root selector is a pseudo-class that represents the root element of the document. In most cases, the root element is the html element.
  2. When you use :root in your CSS, you can define global styles that apply to the entire document or webpage. This is particularly useful for setting variables or defining CSS custom properties (also known as CSS variables) that can be used throughout your CSS code.

scroll behaviour in css

  1. n CSS, the scroll-behavior property is used to control the scrolling behavior of an element with overflow content (such as a container with scrollable content). It allows you to define whether the scrolling should be smooth or instant.

object-fit property

  1. The object-fit property in CSS is used to control how an image or video is scaled and positioned within its container element. It allows you to define how the media element should be fitted into the available space while preserving its aspect ratio.
  2. The object-fit property can be applied to
    • img
    • video
    • svg tags
  3. The possible values for the object-fit property are:
    • fill: This is the default value. The image or video will completely fill its container, potentially stretching or squishing it to fit both the width and height of the container. This may cause the media to lose its original aspect ratio.
    • contain: The image or video will be scaled proportionally to fit entirely within its container while maintaining its aspect ratio. It will be centered within the container, and there may be empty spaces around the media if the container's aspect ratio is different from the media's aspect ratio.
    • cover: The image or video will be scaled proportionally to cover the entire container while maintaining its aspect ratio. It may overflow the container, and some parts of the media might be cropped.
    • none: The image or video will be displayed at its original size, and it may overflow the container if it's larger than the container.
    • scale-down: The image or video will be scaled down proportionally to fit its container while maintaining its aspect ratio. It behaves like contain, but if the media's natural size is smaller than the container, it will be displayed at its natural size.

header tag

  1. It is typically used to define the header section of a page, which often includes elements like logos, navigation menus, headings, and other essential content that appears at the top of the page or the top of a specific section

overflow

  1. "overflow" refers to how content that overflows the dimensions of a container element should be handled. It is used to control the behavior of content when it exceeds the allocated space within a box.

z index

  1. In CSS, the z-index property is used to control the stacking order of positioned elements, particularly when they overlap each other. It determines which elements appear in front of or behind other elements on the z-axis (depth axis), similar to how the "depth" works in a 3D space.

backdrop-filter

  1. The backdrop-filter property in CSS is used to apply visual effects, such as blurring or color shifting, to the area behind an element. It allows you to create interesting effects by altering the appearance of the background that is "backdropped" by a particular element, without affecting the content of that element itself.
